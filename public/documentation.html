Documentation Page

Business logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title /movies/titles/[Title] GET None A JSON object holding data about a single movie, containing movie description, genre, director, image URL, and whether it is featured or not. Example: { Title: “Gladiator”, Movie_description: “A former Roman General sets out to exact vengeance against the corrupt emperor who murdered his family and sent him into slavery.”, Genre: [“Action”, “Drama”], Director: “Ridley Scott”, Image: “http://localhost:3000/movies/images/gladiator”, Featured: yes }
Get list of movies in a genre /movies/genres/[genre] GET None A JSON object holding data about a genre, containing the genre description, and a list of all movies on platform in that genre. Example: { Genre_name: “Thriller”, Thriller_description: “A thriller movie is a genre of film that is designed to keep the audience on the edge of their seats through suspenseful and tense situations. Thriller movies often involve a plot that is centered around a crime, mystery, or danger, with the main character or characters facing intense physical or psychological challenges.”, Thriller_movies: “Butterfly Effect”, “Get out”, “The Prestige”... }
Get data about a director /movies/directors/[director] GET None A JSON object holding data about a director, containing the bio, date of birth, and death year. Example: { Director_name: “Ridley Scott”, Bio: “Ridley Scott is a highly acclaimed English film director and producer. Scott began his career directing commercials and made his feature film debut with "The Duellists" in 1977. He then directed the science fiction classic "Alien" in 1979, which catapulted him to international success. Scott is known for his visually stunning and atmospheric films, as well as his ability to create compelling characters and tell gripping stories. He continues to be one of the most respected and influential filmmakers in the industry.”, DOB: “November 30, 1937”, Death: “-” }
Add a user /users POST A JSON object holding data about the user to add, structured like: { name: “Rachel Lawrie”, username:”r.lawrie”, password: “13cat333plate&%” } A JSON object holding data about the user that was added. Example: { id: 2380-293-2333, name: “Rachel Lawrie”, username:”r.lawrie”, password: “13cat333plate&%” }
Update username /users/[Username]/[New username] PUT None A text message indicating the name of the user and the successful updating of the username.
Update password /users/passwords/[Password] PUT None A text message indicating the username and the successful updating of the password.
Add a movie to favorites list /users/favorites/[Username]/[movie title] POST None A text message indicating [title of movie] has been added to “Favorites.”
Delete a movie from favorites list /users/favorites/[Username]/[movie title] DELETE None A text message indicating [title of movie] has been deleted from “Favorites.”
Deregister /users/[Username] DELETE None A text message that says, “[Username] was successfully deregistered.”
